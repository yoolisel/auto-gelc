# auto-gelc

## requirements
- selenium
- chrome web driver

## how to use
1. fill the variables on the top of the code.
2. run python
```batch
python __main__.py
```
## disclaimer

```
# options.add_arguments("--headless")
```

you have to edit a little to use multiple lectures.
